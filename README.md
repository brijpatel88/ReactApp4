# Movie & TV Series Voting App

A modern React voting application built for React Development Assignment 4.

This application allows users to vote on their favorite movies and TV series using reusable components, global state management, and unit testing.

## Features

- Vote for movies and TV series
- Remove votes
- Reset all votes
- Search by title
- Filter by category
- Current winner tracking
- Movie posters and modern UI
- Responsive design
- Unit testing with Vitest

## Technologies Used

- React
- TypeScript
- Vite
- Context API
- useReducer
- Vitest
- React Testing Library
- CSS
